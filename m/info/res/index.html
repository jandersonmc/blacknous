1 Sistemas Operacionais 💻

1.1 Conceitos Fundamentais

  • Hardware vs Software: Hardware é a parte física (o que você chuta). Software é a parte lógica (o que você xinga).
  • Firmware: Software embutido no hardware (ex: BIOS da placa-mãe).
  • Kernel (Núcleo): Coração do Sistema Operacional. É a camada de software mais próxima do hardware, gerenciando memória, processador e dispositivos.
  • Shell (Interpretador): É a interface entre o usuário e o Kernel. Quando você digita um comando no prompt ou clica num ícone, o Shell traduz isso para o Kernel.

✏️ Notas e Dicas do Professor

1.2 Windows 10: Arquiteturas 32 e 64 bits

  • 32 bits (x86): A arquitetura de 32 bits permite gerenciar, no máximo, cerca de 4 GB de memória RAM. Um único programa só pode usar até 2 GB.
  • 64 bits (x64): Arquitetura moderna. Permite gerenciar virtualmente até 16 Exabytes de RAM. Mais velocidade e estabilidade.
  • Compatibilidade: O Windows 64 bits roda normalmente programas antigos de 32 bits. Porém, um Windows 32 bits NÃO roda programas de 64 bits.

1.3 Windows 10: Recursos Nativos

  • Windows Defender: O antivírus nativo da Microsoft. Hoje chamado de Segurança do Windows. Evoluiu e possui excelente proteção contra malwares e ransomwares.
  • BitLocker: Ferramenta de criptografia de disco. Protege os dados caso o notebook ou HD seja roubado. (Disponível nas versões Pro e Enterprise).
  • OneDrive: Nuvem da Microsoft fortemente integrada ao Windows 10. Permite Salvar Automático e acesso via Explorador de Arquivos.
  • Cortana: Assistente virtual de produtividade (usa comandos de voz ou texto).

1.4 Windows 10: Atalhos de Teclado de Ouro

A tecla com o símbolo do Windows se chama WinKey (Win).

  • Win + D: Mostra ou oculta a Desktop (Área de Trabalho).
  • Win + E: Abre o Explorador de Arquivos.
  • Win + L: Bloqueia (Lock) o computador ou troca de usuário.
  • Win + I: Abre as Configurações do Windows (a engrenagem).
  • Win + R: Abre a caixa de diálogo Executar (Run).
  • Alt + Tab: Alterna rapidamente entre as janelas abertas.

✏️ Notas e Dicas do Professor

1.5 Linux (SUSE): Princípios

  • Open Source: Código-fonte aberto. Qualquer um pode ver, modificar e distribuir. Diferente do Windows (fechado/proprietário).
  • Case Sensitive: O Linux diferencia Maiúsculas de Minúsculas. O arquivo Prova.pdf é totalmente diferente de prova.pdf.
  • Multitarefa Preemptiva e Multiusuário: Vários usuários podem usar recursos simultaneamente sem travar o sistema.
  • Distribuições (Distros): O Kernel do Linux é comum a todos, mas as ferramentas e interfaces variam. O SUSE é uma distro focada no mundo corporativo.

1.6 Linux: Estrutura de Diretórios (FHS)

No Linux, não existe C: ou D:. Tudo começa na Raiz (/).

  • / : O diretório RAIZ. Tudo está dentro dele.
  • /home : Onde ficam os arquivos pessoais dos usuários comuns (Documentos, Downloads).
  • /root : Diretório pessoal do Superusuário (Administrador supremo).
  • /etc : Arquivos de configuração global do sistema (não há executáveis aqui).
  • /bin : Binários (executáveis) essenciais e comandos básicos (como ls, cp).
  • /var : Arquivos variáveis (arquivos de log, spool de impressão, e-mails).

1.7 Linux: Comandos Essenciais

  • ls: Listar arquivos e diretórios.
  • cd: Change Directory (mudar de pasta).
  • pwd: Mostra o caminho da pasta atual onde você está.
  • cp: Copiar arquivos ou pastas.
  • mv: Mover (recortar) ou Renomear arquivos.
  • rm: Remover (apagar) arquivos. Atenção: não vai para a lixeira no shell!
  • chmod: Altera as permissões de acesso (Leitura, Gravação, Execução).

2 Edição de Textos, Planilhas e Apresentações 📝

2.1 Word: Conceitos Gerais

  • Formato Padrão: .docx (formato XML compactado, mais leve e seguro).
  • Página Inicial: É a guia mais usada. Contém formatação de Fonte (Negrito, Itálico, Cor), Parágrafo (Alinhamento, Espaçamento) e Estilos.

2.2 Word: Atalhos Brasileiros (Cai muito!)

No Brasil (Word em PT-BR), os atalhos são traduzidos. Cuidado com o Ctrl+T!

  • Ctrl + U: Substituir palavras.
  • Ctrl + N: Negrito.
  • Ctrl + S: Sublinhado.
  • Ctrl + J: Justificar o texto (alinhar bordas esquerda e direita).

✏️ Notas e Dicas do Professor

2.3 Excel: Fundamentos das Fórmulas

  • Toda e qualquer fórmula ou função deve começar com o sinal de igual (=).
  • Referência Relativa (Ex: A1): Se você copiar a fórmula para a linha de baixo, ela muda sozinha para A2.
  • Referência Absoluta (Ex: $A$1): O cifrão ($) TRAVA a referência. Se você arrastar a fórmula, ela continuará apontando fixa para A1. Pode travar só a coluna ($A1) ou só a linha (A$1).
  • Dois pontos ( : ) vs Ponto e vírgula ( ; ):
    • A1:A5 significa "De A1 ATÉ A5" (Intervalo contínuo).
    • A1;A5 significa "A1 E A5" (Células isoladas).

2.4 Excel: Funções Básicas

  • =SOMA(A1:A5): Soma todos os valores no intervalo.
  • =MÉDIA(A1:A5): Calcula a média aritmética. Ignora células com texto.
  • =MÁXIMO(A1:A5): Retorna o maior valor numérico do intervalo.
  • =MÍNIMO(A1:A5): Retorna o menor valor numérico do intervalo.
  • =MAIOR(A1:A5; 2): Retorna o 2º maior valor do intervalo.

✏️ Notas e Dicas do Professor

2.5 Excel: Funções Avançadas e Lógicas

  • =SE(teste_logico; valor_se_verdadeiro; valor_se_falso): Retorna um valor se a condição for V, e outro se for F. Ex: =SE(A1>=7; "Aprovado"; "Reprovado").
  • =PROCV(valor_procurado; matriz_tabela; num_indice_coluna; [procurar_intervalo]): Procura um valor na primeira coluna à vertical, e retorna o valor da linha correspondente.
  • =CONT.NÚM(A1:A10): Conta apenas as células que possuem números.
  • =CONT.VALORES(A1:A10): Conta todas as células que não estão vazias (tem texto, número, erro).

2.6 PowerPoint: Apresentações e Atalhos

  • Extensão Padrão: .pptx. Se salvo como .ppsx, o arquivo abre diretamente no modo Apresentação, sem abrir a tela de edição.
  • Slide Mestre: Funciona como um "molde". Tudo que você insere nele (logotipo, fontes, fundos) é aplicado automaticamente a todos os slides. Ideal para padronização.
  • Transição vs Animação: Transição é o efeito de passagem de um slide inteiro para o outro. Animação é o efeito dado aos objetos DENTRO do slide (textos, imagens).
  • Atalhos: F5 inicia a apresentação do 1º slide. Shift + F5 inicia a partir do slide atual.

3 Redes de Computadores e Internet 🌐

3.1 Abrangência das Redes (Geográfica)

  • PAN (Personal Area Network): Rede pessoal, metros de distância. Ex: Bluetooth entre celular e fone.
  • LAN (Local Area Network): Rede Local. Mesmo prédio ou campus. Alta velocidade.
  • MAN (Metropolitan Area Network): Rede Metropolitana. Conecta filiais na mesma cidade.
  • WAN (Wide Area Network): Rede de Longa Distância. Cobre países ou continentes. A Internet é a maior WAN.

✏️ Notas e Dicas do Professor

3.2 Topologias Físicas

  • Estrela: Todos os computadores se conectam a um equipamento central (Switch). Se um cabo rompe, só aquele PC cai. A mais usada.
  • Barramento (Bus): Todos usam um cabo central compartilhado. Se o cabo rompe, a rede cai toda.
  • Anel (Ring): Conectados em círculo. O sinal passa de máquina em máquina.
  • Malha (Mesh): Todos conectam com todos. Alta redundância. Muito usada na internet (roteadores).

3.3 Equipamentos de Rede

  • Switch (Comutador): Liga PCs em LAN (Estrela). Ele entrega dados apenas para o destinatário correto usando endereço MAC (Camada 2).
  • Roteador (Router): Conecta redes DIFERENTES (Ex: sua LAN à Internet). Escolhe a melhor rota usando IP (Camada 3).
  • Access Point (AP): Ponto de acesso sem fio (Wi-Fi).

3.4 Protocolos Básicos: TCP/IP, IP, DHCP e DNS

  • IP (Internet Protocol): É o "CPF" da máquina. IPv4 tem 32 bits. IPv6 tem 128 bits.
  • DNS (Domain Name System): "Lista telefônica". Transforma nomes fáceis (google.com) em IPs numéricos.
  • DHCP: Entrega endereços IP dinamicamente para as máquinas quando elas se conectam na rede.
  • TCP vs UDP: TCP garante a entrega (orientado a conexão). UDP é rápido, mas não garante a entrega (usado em streaming e jogos).

✏️ Notas e Dicas do Professor

3.5 Internet, Intranet e Extranet

  • Internet: Rede global e pública.
  • Intranet: Rede PRIVADA corporativa. Usa os mesmos protocolos (TCP/IP) da Internet! A diferença é o acesso restrito.
  • Extranet: Parte da Intranet disponibilizada para usuários externos (parceiros/clientes VIP), mediante login via internet.

4.3 Navegação Privativa / InPrivate

No Edge (Ctrl+Shift+N), no Firefox (Ctrl+Shift+P).

  • O QUE FAZ: Apaga Histórico, Cookies e Senhas da sessão no seu computador ao fechar.
  • O QUE NÃO FAZ: Ela NÃO oculta sua navegação do seu provedor, chefe ou sites. Não é VPN! Também NÃO deleta os arquivos baixados.

4.4 Google: Busca Avançada

5 Correio Eletrônico (E-mail) ✉️

5.1 Cabeçalho: Para, Cc e Cco

  • Para (To): Destinatários principais.
  • Cc (Com Cópia): Destinatários secundários. Todos veem os endereços.
  • Cco ou Bcc (Com Cópia Oculta): Oculta QUEM ESTÁ NO CCO. Quem está no Para/Cc não vê quem está no Cco. Quem está no Cco vê o resto.

5.2 Responder vs Responder a Todos

  • Responder: A resposta vai APENAS para o remetente original.
  • Responder a Todos: Resposta vai para Remetente + 'Para' + 'Cc'.
  • Atenção: Quem estava em Cco JAMAIS recebe a "resposta a todos" para não ser revelado.

✏️ Notas e Dicas do Professor

5.3 Protocolos de E-mail

  • SMTP (Envio): Sua Mensagem Tá Partindo. Usado para ENVIAR. Usa porta 587 ou 465 (criptografada).
  • POP3 (Receber/Puxar): Faz o download e APAGA do servidor. Para ler offline em 1 PC.
  • IMAP (Receber/Sincronizar): Acessa Múltiplos. Mantém e-mails no servidor e sincroniza pastas.

5.4 Webmail vs Cliente de E-mail

  • Webmail: Acesso pelo Navegador (gmail.com). Sem instalar apps.
  • Cliente: Programa instalado localmente (Outlook, Thunderbird). Usam SMTP/IMAP ativamente.

7 Segurança da Informação e Backups 🛡️

7.1 Princípios Básicos (D.I.C.A)

  • Disponibilidade: O sistema DEVE estar acessível.
  • Integridade: Dados não podem ser alterados indevidamente.
  • Confidencialidade: Sigilo absoluto. Acesso só autorizado. Garantida pela Criptografia.
  • Autenticidade: Prova a identidade do autor (Não-repúdio). Garantida pela Assinatura Digital.

7.2 Vírus vs Worms

  • Vírus: PRECISA de arquivo hospedeiro e PRECISA ser executado (clicado) pela vítima.
  • Worm (Verme): AUTÔNOMO. NÃO precisa de hospedeiro. Se autorreplica explorando falhas de rede e causa lentidão na rede.

✏️ Notas e Dicas do Professor

7.3 Ransomware e Trojan

  • Ransomware: O sequestrador digital! Ele criptografa todos os dados do HD e cobra um resgate (geralmente em Bitcoin).
  • Trojan (Cavalo de Troia): Disfarce. Parece um programa legítimo, mas por trás abre portas (Backdoors) para invasores.

7.4 Spyware, Botnet e Phishing

  • Spyware: Programa espião. Ex: Keylogger (captura teclado) e Screenlogger (captura tela).
  • Botnet: Rede de computadores zumbis usados para atacar servidores (Ataque DDoS).
  • Phishing: Golpe de Engenharia Social. E-mails e sites falsos tentando roubar senhas ("pescador de dados").

7.5 Ferramentas de Segurança

  • Criptografia Simétrica: Mesma chave para cifrar e decifrar (rápida).
  • Criptografia Assimétrica: Par de chaves (Pública e Privada). Mais segura.
  • Assinatura Digital: Garante Autenticidade e Integridade usando HASH.
  • Firewall: Filtra o TRÁFEGO de rede (Regras IP/Porta). Atenção: Firewall NÃO escaneia/apaga vírus!
  • Antivírus: Analisa arquivos buscando código malicioso e os coloca em quarentena.

✏️ Notas e Dicas do Professor

7.6 Backups e Nuvem

  • Backup Completo (Full): Demora para fazer, muito rápido para restaurar.
  • Backup Incremental: Copia o que mudou desde o ÚLTIMO BACKUP. Rápido para fazer, demorado para restaurar (junta todas as partes).
  • Backup Diferencial: Copia o que mudou desde o último Backup COMPLETO.
  • Cloud Storage: Nuvem (Google Drive, OneDrive). Garante redundância de dados (não perde se o PC queimar).
